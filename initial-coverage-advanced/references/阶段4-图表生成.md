# 阶段 4：图表生成

产物：`[公司]_图表_[日期].zip`，含 40–70 张 PNG（300 DPI）+ 一份图表清单（文件名 + 标题 + 数据来源 + 对应正文章节）。

前置：阶段 1、2、3 产物齐，外部行情数据（股价、可比公司倍数）能获取。缺失则停下补充。

中文深度研报图表密度极高（标杆 44–87 张），图和表统一连续编号。这一阶段把前三阶段的数据可视化，阶段 5 全部嵌入正文。

## 技术栈

Python + matplotlib + pandas。两件事必须先解决：

**中文字体**（不配置会导致中文显示为方框乱码）。按平台选择可用的中文字体：

```python
import matplotlib
import matplotlib.pyplot as plt
# macOS 优先级：Songti SC / PingFang SC / Arial Unicode MS
# 也可指定思源宋体/黑体的 ttf 路径
matplotlib.rcParams['font.sans-serif'] = ['Songti SC', 'PingFang SC', 'Arial Unicode MS', 'Heiti SC']
matplotlib.rcParams['axes.unicode_minus'] = False  # 负号正常显示
```

生成第一张图后**务必查看一次**，确认中文不是方框，再批量生成。

**研报配色**。全报告一套主色，贴合券商风格（中泰红系 / 华创蓝系皆可）。定一个主色 + 辅助色板：

```python
# 示例：红系主色板
PALETTE = ['#C0202E', '#E08A1E', '#7F7F7F', '#1F4E79', '#9DC3E6']
# 多分部堆叠/对比时按顺序取色，保持全报告一致
```

## 图表规范

每张图：

- **标题**「图表 N：[描述] [单位]」，图内或图上方
- **来源**小字脚注「来源：公司公告、WIND、XX研究所」
- 300 DPI（`dpi=300`），尺寸适配 A4 单栏或双栏
- 文件名带序号便于阶段 5 按序嵌入，如 `chart_11_分部营收.png`
- 数据标签清晰，不重叠、不超出边界、不裁切；坐标轴有单位

### 坐标轴与边框

直角坐标图（柱状图、折线图、面积图、散点图）统一去掉上、右边框线，保留 X / Y 实线；所有图一律不画网格线：

```python
ax.spines['top'].set_visible(False)
ax.spines['right'].set_visible(False)
ax.spines['left'].set_visible(True)     # X / Y 实线保留
ax.spines['bottom'].set_visible(True)
ax.grid(False)                          # 全部图去网格线
```

上边线和右边线永远不要显示。饼图、雷达图、热力图、足球场图、瀑布图没有常规直角坐标轴，按各自惯例处理，不强套这条。

### 双 Y 轴（股价走势对比图）

个股股价和指数点位量纲差几十倍，放同一条 Y 轴个股线会被压平。用 `twinx()` 双轴，两条线都画绝对值、都按周频：

```python
fig, ax = plt.subplots()
ax.plot(weeks, stock_price, color=PALETTE[0], label='个股股价（元）')   # 左轴：个股绝对股价
ax2 = ax.twinx()
ax2.plot(weeks, index_level, color=PALETTE[3], label='指数（点）')      # 右轴：指数绝对点位
ax.set_ylabel('股价（元）'); ax2.set_ylabel('指数（点）')
# 左右轴各自图例，或合并图例；ax2 同样去掉 top spine
ax2.spines['top'].set_visible(False)
```

不要把任一序列归一化到 100。

### 双图并列尺寸

成对/可对照的图（分部增速对比、可比公司倍数、三情景、毛利率 vs 净利率等）在阶段 5 会一行两图各占 50% 栏宽并列。这类图按窄版生成——长宽比偏 4:3、`figsize` 收窄、字号适当放大，避免缩到 50% 后文字发虚发小。其余单栏全宽图按常规尺寸。阶段 5 用无边框二列表格放置（见 `Word文档格式标准.md`）。

## 图表清单（按数据来源映射）

40–70 张，覆盖前三阶段 + 外部数据。下面是骨架，按公司和行业增减。

**必备图（不可缺）**：

- 股价走势对比（个股 vs 指数，近 12 个月，封面用）：**双 Y 轴**——左轴个股绝对股价、右轴指数绝对点位，两者都是绝对值，**严禁指数化 / 归一化到 100**；**周频取样**（约 52 个点）
- 分部营业收入（堆叠柱或堆叠面积，历史 + 预测）
- 盈利预测（收入/归母净利/增速，历史 + 5 年预测）
- 估值：SOTP 拆解图（多业务）或可比公司倍数对比
- 估值足球场图（各方法估值区间 + 当前价 + 目标价）
- DCF 敏感性热力图（若做了 DCF）

**来自阶段 1（公司研究）**：

- 历史沿革时间轴
- 股权结构图
- 组织架构图
- 分部收入占比（饼图或环形）
- 分部毛利率对比
- 费用率与净利率趋势
- 行业市场规模（历史 + 预测 CAGR）
- 产业链图
- 市占率对比
- 竞品对标（雷达图或对比柱）
- 管理层/核心团队（可做表）

**来自阶段 2（盈利预测模型）**：

- 分部营收预测（堆叠）
- 分部增速对比
- 毛利率/净利率预测趋势
- 三情景对比（乐观/中性/悲观的收入或利润路径）
- 历史回测对照（模型回算 vs 实际）
- 资本开支/研发投入趋势
- 现金流趋势

**来自阶段 3（估值）**：

- 可比公司倍数散点或对比柱
- 可比公司分位统计
- SOTP 分部估值瀑布/堆叠
- DCF 现金流折现瀑布
- DCF 敏感性热力图
- 历史估值通道（PE/PB 历史分位）
- 足球场图

**外部数据**：

- 股价走势对比
- 历史估值通道（需历史股价 + 历史 EPS/BPS）
