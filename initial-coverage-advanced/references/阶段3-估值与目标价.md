# 阶段 3：估值与目标价

产物：估值稿 `[公司]_估值分析_[日期].md`（3–5 页）+ 估值页并入阶段 2 的同一个 Excel（不另起文件）。给出目标价、评级、上行空间、催化剂。

前置：阶段 2 模型在，且含分部收入预测、利润预测、关键假设。缺失则停下补充。

中文研报的估值以**相对估值 + 分部 SOTP 为主，DCF 辅证**。理论细节看 `估值方法学.md`，这里讲怎么做、怎么落到目标价。

## 选方法：跟着业务结构走

- **单一主业、可比公司多** → 相对估值（PE/PEG 为主，重资产看 EV/EBITDA，金融地产看 PB）打底，DCF 辅证
- **多业务、各块财务特征差异大**（互联网平台、控股集团）→ 分部 SOTP 为主，每个分部用最贴合的方法和倍数，加总得整体估值。标杆里阿里就是 SOTP（电商、云、其他分开估）
- **高成长、当期利润不可比**（扭亏中的科技公司）→ DCF 或 EV/Sales、PS 为主，PE 失效时不要强行套用

至少两种方法交叉验证，不要只靠一种。

## 相对估值

1. 选 5–10 家可比公司：同行业 > 同业务模式 > 同规模 > 同增速 > 同区域，按这个优先级筛
2. 取每家的市场数据（市值、EV）和财务数据（收入、净利、EBITDA），算倍数（PE、PEG、EV/EBITDA、EV/Sales、PB，按行业选）
3. 做**分位统计**：最大 / 75 分位 / 中位 / 25 分位 / 最小。这是研报惯例，给目标公司定倍数时落在分布里有依据，不是主观臆定
4. 给目标公司选倍数（参照中位或按成长性/壁垒给溢价折价，溢价折价要讲理由），× 阶段 2 的预测利润 = 估值

配可比公司估值表、倍数对比图。

## 分部 SOTP

多业务公司的主力方法：

1. 按阶段 1/2 的分部划分，每个分部单独估——成熟现金牛业务用 PE 或 EV/EBITDA，高成长业务用 PS 或 DCF，投资类资产按公允价值或账面打折
2. 每个分部的方法和倍数都要有依据（对标该业务的纯粹可比公司）
3. 加总各分部价值，减净负债、减控股折价（如适用）= 整体股权价值
4. 除以股本 = 每股 SOTP 价值

配 SOTP 拆解图（瀑布图或堆叠条，展示各分部贡献）。

## DCF 辅证

不作主方法时也值得做一版佐证目标价区间：

1. 取阶段 2 的收入、利润预测，推自由现金流（FCFF）
2. WACC：CAPM 算权益成本（无风险利率 + β × 风险溢价）+ 债务成本，按资本结构加权
3. 终值：永续增长法（永续增速 2–3%）或退出倍数法
4. 折现到现值，加总得 EV，减净负债得股权价值
5. **敏感性表**：WACC × 永续增长 的目标价矩阵（配热力图），看目标价对核心假设多敏感

DCF 的假设要和阶段 2 模型联动，不是另建一套。

## 落到目标价和评级

1. 几种方法的结果汇总，给一个目标价（或加权，或取主方法、其余佐证），写清怎么得来的
2. 上行空间 = （目标价 − 当前价）/ 当前价，标当前价日期
3. **评级（首次）**：按上行空间和所在市场惯用档给——A 股常用买入/增持/中性/减持/卖出，港股美股可用对应档。首次覆盖标"（首次）"
4. **催化剂** 3–5 条：什么事件会兑现这个判断（业绩超预期、新品放量、政策落地、订单签约），尽量带时间
5. **足球场图**：横向条形，把各方法的估值区间（相对估值/SOTP/DCF）和当前价、目标价画在同一标尺上，直观呈现目标价的位置

## Excel 估值页

并入阶段 2 的 Excel，新增几页：

- 相对估值页（可比公司 + 分位统计）
- SOTP 页（分部估值加总）
- DCF 页（FCF 折现 + 敏感性矩阵）
- 估值汇总页（各方法结果 + 目标价 + 评级，跨表引用模型页的利润数，绿色字）

所有利润、每股数引用阶段 2 模型页，不抄录数值。recalc 零错误。

## 自检

- 至少两种方法，结果能相互印证（差异过大要解释）
- 可比公司分位统计齐
- 目标价能从方法回溯，不是主观臆定
- 估值用的利润/每股数和阶段 2 模型一致
- 评级、目标价、上行空间、催化剂齐
- 当前价标了日期
