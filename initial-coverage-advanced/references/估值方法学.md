# 估值方法学

阶段 3 的理论参考。讲清每种方法的原理、参数怎么定、常见错误在哪。具体操作流程看 `阶段3-估值与目标价.md`，这份是需要时查阅的详细资料。

## 相对估值

核心是"同类资产同价"——找一组可比公司，看市场给它们什么倍数，套用到目标公司。

**常用倍数与适用场景**：

| 倍数 | 算法 | 适用 |
|------|------|------|
| PE | 市值 / 净利润 | 盈利稳定的成熟公司 |
| PEG | PE / 净利增速 | 成长股，剔除增速差异后比较 |
| EV/EBITDA | 企业价值 / EBITDA | 重资产、资本结构差异大的公司，剔除折旧和杠杆影响 |
| EV/Sales | 企业价值 / 收入 | 未盈利或利润不可比的高成长公司 |
| PB | 市值 / 净资产 | 银行、保险、地产等净资产驱动的行业 |
| PS | 市值 / 收入 | 软件、SaaS 等早期高成长 |

**可比公司筛选优先级**：同行业 > 同业务模式 > 同规模 > 同增速 > 同区域。选 5–10 家，太少没统计意义，太多稀释可比性。

**分位统计**：算可比组的最大/75 分位/中位/25 分位/最小。给目标公司定倍数时落在这个分布里，溢价或折价相对中位来定，并讲清理由（更高成长、更强壁垒给溢价；治理或周期风险给折价）。

**常见错误**：可比公司选得不可比（用不同商业模式勉强凑数）；用历史倍数套未来利润而不调整；PE 在亏损或微利时失效仍强行使用；忽略一次性损益让利润口径不可比。

## 分部加总 SOTP

把公司拆成几块各自估值再加总，适合多业务、各块财务特征差异大的公司（互联网平台、控股集团、多元化集团）。

**步骤**：每个分部找各自的纯粹可比公司，用最贴合的方法和倍数估值 → 加总各分部价值得企业价值 → 减净负债 → 减控股折价（如有少数股东或集团折价）→ 得股权价值 → 除股本得每股价值。

**关键判断**：
- 成熟现金牛分部用 PE 或 EV/EBITDA；高成长分部用 PS 或 DCF；投资类资产（持有的上市公司股权、长期投资）按公允价值或账面打折计入
- 控股折价：集团层面通常给一定折价（10–20% 常见），反映分部协同不足或治理摩擦，要说明依据
- 避免重复计算：分部间的内部交易、母公司现金不要算两遍

**常见错误**：给每个分部都用同一个倍数（违背了分部估值的初衷）；投资类资产按账面而非市值计；漏掉净负债或控股折价。

## DCF（现金流折现）

绝对估值，把未来自由现金流折现到今天。理论上最严谨，但对假设极敏感，研报里多作佐证。

**步骤**：

1. **预测自由现金流（FCFF）**：FCFF = 税后经营利润（EBIT×(1−税率)）+ 折旧摊销 − 资本开支 − 营运资本增加。取阶段 2 模型的收入、利润、资本开支预测，不要另建一套
2. **算 WACC**：
   - 权益成本 Re = 无风险利率 Rf + β × 市场风险溢价（CAPM）
   - 债务成本 Rd = 实际融资利率 ×(1−税率)
   - WACC = Re ×(E/V) + Rd ×(D/V)，按市值口径的资本结构加权
   - 典型区间：成熟公司 8–10%，高成长/高风险 10–14%
3. **终值**：
   - 永续增长法：终值 = FCFF(末年+1) /(WACC − g)，永续增速 g 取 2–3%（不能高于长期 GDP 增速）
   - 退出倍数法：终值 = 末年 EBITDA × 退出 EV/EBITDA 倍数
   - 两种都算，交叉验证
4. **折现求和**：各年 FCFF 和终值按 WACC 折现到现值，加总得企业价值 EV → 减净负债 → 股权价值 → 除股本 → 每股内在价值
5. **敏感性分析**：做 WACC × 永续增长（或 WACC × 退出倍数）的目标价矩阵，配热力图。终值通常占 DCF 估值的 60–80%，所以这两个参数的微小变化会大幅移动结果，必须展示敏感性

**常见错误**：永续增速定太高（超过 GDP 长期增速）；WACC 凭主观设定、不拆解；FCF 和盈利模型口径不一致；只给一个点估值不给敏感性区间；终值占比过高还不做压力测试。

## 可比交易（如适用）

并购或一级市场定价时参考——看同行业近期成交案例的估值倍数。受当时市场情绪、协同溢价、控制权溢价影响，波动大，公开股权研报里用得少，作补充参照。

## 方法间怎么取舍

- 互联网平台、控股集团 → SOTP 主，相对估值和 DCF 辅
- 盈利稳定的成熟公司 → 相对估值（PE）主，DCF 辅
- 高成长扭亏中的科技公司 → DCF 或 PS/EV-Sales 主，待盈利稳定后切回 PE
- 周期股 → 看周期位置，PB 或正常化盈利的 PE，避免在周期高点用峰值利润套用 PE

目标价可取主方法结果，或几种方法加权，但都要写清怎么得来。把各方法的估值区间画进足球场图，让读者看到目标价的位置和分歧。
